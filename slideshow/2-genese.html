Identification d'un film

Pour identifier un film, nous nous sommes rendu compte
que le titre ne suffisait pas. (O)

Ici on peut voir 2 affiches de films ayant le même titre "LIFE"
et il en existe d'autres.

Il fallait donc trouver une donnée liée au film qui l'identifierait à coup sûr. (O)

Solutions trouvées

  • le n° ISAN
  • le code IMDB
Les solution que nous avons pu trouver, c'est (O)

Le n° ISAN pour International Standard Audiovisual Number
qui est un numéro unique,
universel et permanent d'immatriculation
des œuvres audiovisuelles. (O)

Le code IMDB pour Internet Movie Database qui est une base
de données en ligne sur le cinéma mondial, sur la télévision,
et plus secondairement les jeux vidéo. (O)

The Open Movie Database

The Open Movie Database contient beaucoup d'informations intérressantes comme :
  • le Titre évidemment
  • le synopsis
  • le "IMDB Rating"
  • le "IMDB votes"
(O)
Mais elle sont toutes en Anglais
Difficile de traduire, dans l'application, un synopsis pour des utilisateurs français. (O)

The Movie Database

The Movie Database contient des informations en plusieurs langues
(FR, EN, DE, ...)
L'API est bien plus complète
Et surtout l'API est sans restrictions
(O)
Cerise sur le gâteau, l'API se base sur le n° IMDB
qui identifie la grande majorité des films.
Mais, il lui manquent des données interressantes comme les votes et notes IMDB.(O)

La solution

Récupérer les informations grâce au n° IMDB sur 2 API :
  • The Movie Database
    • le titre
    • l'affiche
